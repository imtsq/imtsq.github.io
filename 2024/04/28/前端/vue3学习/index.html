1.vue3 vs vue2

其实差不多，关键vue3看起来简洁（组合式API开发），快速

setup

<script setup>告诉 Vue 需要在编译时进行一些处理，让我们可以更简洁地使用组合式 API。比如，<script setup> 中的导入和顶层变量/函数都能够在模板中直接使用。

声明式渲染：

声明式渲染：通过扩展于标准 HTML 的模板语法，我们可以根据 JavaScript 的状态来描述 HTML 应该是什么样子的。当状态改变时，HTML 会自动更新。

响应式

改变某个变量，dom会自动渲染html，即动态变化页面值

其中vue3中有两种方法，reactive() 和 ref()其中reactive()只适用于对象 (包括数组和内置类型，如 Map 和 Set)。而另一个 API ref() 则可以接受任何值类型。ref 会返回一个包裹对象，并在 .value 属性下暴露内部值。

vue3语法糖

  • reactive()使用

  • ref()使用，无限制，但是被包裹了，需要.value才能访问其值
